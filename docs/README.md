#### InputView
- [ ] 자판기가 보유하고 있는 금액 입력
- [ ] 상품명, 가격, 수량 입력
  - [ ] 100원 이하면 예외 출력 후 재입력
  - [ ] 10원으로 나누어 떨어지지 않으면 예외 출력 후 재입력
- [ ] 투입 금액 입력

#### OutputView
- [ ] 자판기가 보유하고 있는 동전 목록 출력
  - [ ] 최소 개수 동전으로 잔돈 출력
  - [ ] 지폐로 반환하는 경우는 없음 (500, 100, 50, 10)
- [ ] 반환된 동전만 출력

#### Coin
- [ ] 자판기 보유 동전 개수
- [ ] 잔돈 동전 개수

#### product
- [ ] 상품명, 가격, 수량 목록
- [ ] 상품 최저 가격 


#### vendingMachine
- [ ] 전체적인 자판기 로직
